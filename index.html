Short website development tutorials

Creating a simple web page: intro Storing your code in GitHub: intro

Web pages can be created entirely by writing 'code' into a file. Try the following:

  1. Open a text editor such as Notepad or TextEdit
  2. At the top of your page, type these characters:  <html> 
  3. Type in some information that you'd like to have on your web page, such as your life story
  4. At the end of your text, after the full-stop, type these characters:  </html> 
  5. Save the file, selecting the 'html' file format, and take notice of where you are saving the new file to
  6. Close the text editor
  7. Navigate to where you just saved the file, and click on the file to open it

You should now see your web page, complete with the words you added. This could theoretically be uploaded to the internet in it's current format, however there is still some more 'code' required in order for it to look presentable. The code will mostly consist of 'tags', similar to the  <html>  code that you already created. See the video tutorial here.

Many web developers keep the code for their web pages stored in a place called GitHub.

GitHub (and similar Git websites such as Bitbucket) offer free accounts, and one of the best things about them is being able to compare your current web page code to any older versions.

GitHub and Bitbucket can be confusing to use at first, but they are quite powerful once you know the basics, which can be discussed in a later tutorial.

Basically, you will need to download and install some software to your computer, and then you can link it up with your collection of online code files (called a repository).

Once the software is installed, you can edit and update your web page code without needing to open a web browser. Updating your code in this way is often done via your computer's terminal or command-prompt. I intend to make a video tutorial about this in the future.
