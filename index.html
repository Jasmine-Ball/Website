Creating a simple web pageStoring your code in GitHub

Web pages can be created entirely by writing 'code' into a file.
To do this, open a text editor such as Notepad or TextEdit, and type in the words that you'd like to show.
Save the file, selecting the 'html' file format and take notice of where you are saving the new file to.
Close the text editor.
Navigate to where you just saved the file, and click on the file to open it. You should now see your web page, complete with the words you added. This can be uploaded to the internet and will work just fine as it is, however there is some invisible "code" that should be added in order to make it look good an function like a professional web page. You can read about this in the next article.

Many web developers keep the code for their web pages stored in a place called GitHub.
GitHub allows free accounts, and one of the best things about it is being able to compare updates to your website to the original version of your website.
In the beginning, it can be confusing to use, but it is quite powerful once you know the basics.
